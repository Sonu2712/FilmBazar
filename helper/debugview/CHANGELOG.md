Summary
=======
It provides a DebugView which can be attached as a Drawer to any activity.
This is meant to be used only in Debug mode.

It adds following dependency to project for Debug Support:

-> MadgeLayout
-> ProcessPhoenix
-> LeakCanary
-> ThreeTenABP
-> Retrofit Mock
